# Product Focus

## Product
Analyst Pro

## Positioning
A consultant / freelancer analytics toolkit for turning messy client spreadsheets into clean, trustworthy, client-ready reports.

## Ideal Users
- freelance data analysts
- marketing consultants
- operations consultants
- finance/accounting freelancers
- small agencies handling recurring client reports

## Core Workflow
1. Upload messy client spreadsheet
2. Detect structure and clean data
3. Review data health and cleaning actions
4. Generate top insights and charts
5. Compare with previous file
6. Build client-ready report
7. Export polished output

## What We Are Not
- not a generic BI platform
- not a toy AI CSV chatbot
- not an enterprise analytics suite
- not an AutoML-first product

## v1 Core Promise
Upload messy client files and get trustworthy, polished, client-ready analysis fast.
